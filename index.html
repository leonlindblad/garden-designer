How do I get an API key?
  1. Go to Google Cloud Console.
  2. Create a project (or pick one).
  3. Open APIs & Services → Library and enable Maps JavaScript API and Geocoding API.
  4. Go to Credentials → Create Credentials → API key.
  5. (Recommended) Restrict the key to your website address + the two APIs above.
  6. Paste the key above. It stays on your device only.

Your key is stored locally in your browser and sent only to Google. It never touches our server (there is no server).
